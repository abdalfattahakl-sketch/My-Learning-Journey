## 📅 اليوم الأول (Day 1) - HTML5 Essentials
تم بحمد الله إنجاز اليوم الأول والبدء في كورس HTML5.

### 📝 ما تم تعلمه وتطبيقه اليوم:
- فهم الهيكل الأساسي لصفحة الويب والفرق بين عناصر الـ `<head>` والـ `<body>`.
- استخدام أوزان العناوين المختلفة من `<h1>` إلى `<h6>`.
- المقارنة العملية والتنسيق بين الوسوم `<p>` و `<br>` و `<pre>`.
- استكشاف وسم الـ `<meta>` الخاص بالـ refresh وفهم أبعاده في تجربة المستخدم.
- ربط بيئة العمل (VS Code) بحساب GitHub بنجاح.
