# اليوم الرابع: سحر القوائم والعناصر الدلالية (Lists & Semantic HTML)

## 🎯 الأهداف التي تم تحقيقها اليوم:
1. **القوائم البرمجية (Lists):**
   - دراسة القوائم المرتبة `<ol>` وتغيير نوع الترقيم باستخدام خاصية `type`.
   - دراسة القوائم غير المرتبة `<ul>`.
   - دراسة القوائم الوصفية والمصطلحات `<dl>`, `<dt>`, `<dd>` وتوظيفها تربوياً في شرح قواعد النحو.
2. **العناصر الدلالية (Semantic HTML):**
   - تنظيم الهيكل باستخدام: `<header>`, `<nav>`, `<main>`, `<section>`, `<footer>`.
   - فهم أهمية التقسيم لمحركات البحث والـ SEO والاستغناء عن عشوائية الـ `<div>` المتكرر.
3. **الروابط والمسارات (Links & Paths):**
   - ربط 3 صفحات كاملة ببعضها البعض تنقلاً كاملاً.
   - إتقان مفهوم الخروج من المجلدات والدخول إليها باستخدام (`../`) والروابط الذكية للإيميل `mailto:` والهاتف `tel:`.
4. **التخصيص في الـ CSS (Specificity):**
   - حل مشكلة تداخل التنسيقات باستخدام استهداف دقيق لعناصر شريط التنقل (`nav ul li`).

## 📚 المشروع العملي لليوم:
تم بناء موقع **"النحو السالك"** المكون من ثلاث صفحات مترابطة:
- `index.html`: الصفحة الرئيسية للموقع وتحتوي على فهرس الدروس والأهداف.
- `pages/content.html`: صفحة الدرس الأول (أقسام الكلمة) مدعومة بجدول أمثلة واستخدام وسم الـ `<mark>`.
- `pages/contactus.html`: صفحة التواصل المنظمة عبر الجداول والروابط الوظيفية.
