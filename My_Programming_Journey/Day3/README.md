# 📊 Daily Routine & Skill Development Tables

A clean, semantic, and accessible HTML-based dashboard designed to track weekly habits, manage daily tasks, and document programming progress. This project focuses on utilizing advanced HTML table structures, proper data hierarchy, and responsive UI/UX principles.

---

## 🚀 Features & UI/UX Design Implementation

During the development of this dashboard, high emphasis was placed on creating an optimal **User Experience (UX)** and an intuitive **User Interface (UI)**:

*   **Advanced Structural Hierarchy:** Implemented structured data partitioning using semantic tags (`<caption>`, `<thead>`, `<tbody>`, and `<tfoot>`) to ensure excellent readability and search engine optimization.
*   **Complex Cell Merging (`rowspan`):** Applied precise vertical merging (`rowspan="4"`) within the weekly tracker to seamlessly link days with multiple daily habits without visual clutter.
*   **Context-Based Layouts & Alignment:** Used intentional alignment (`align="center"`) and designated fallback indicators (e.g., `-`) for empty rows to maintain interface stability and a balanced layout across various viewports.
*   **Bi-directional Content Flow (`dir`):** Dynamic directional control is enforced across different tables. The habit trackers follow a Right-to-Left (`dir="rtl"`) layout for native Arabic speakers, while the programming dictionary dynamically shifts to a Left-to-Right (`dir="ltr"`) flow to fit coding syntax natively.
*   **Visual Contrast & Breathing Room:** Integrated inline styling and custom vertical spacing (`height="50"`) on table headers alongside specific background colors (like `dodgerblue` and `gray`) to establish a clear visual hierarchy and reduce cognitive load.

---

## 📂 Project Structure Included Tables

1.  **Weekly Habits Tracker (`جدول أسبوعي`):** A macro-view calendar managing 4 essential daily habits (Prayer, Quran, Exercise, Reading) mapped across the 7 days of the week.
2.  **Daily Checklist (`جدول عادات يوم الثلاثاء`):** A micro-view micro-management table focusing on a single day's execution state ("Completed" vs. "Pending").
3.  **Programming Lexicon (`Dictionary for Tuesday`):** A specialized technical dictionary functioning as an immediate lookup table for newly acquired HTML tags, their structural functions, and practical code examples.
